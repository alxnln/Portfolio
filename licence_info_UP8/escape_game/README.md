# Escape The Witch

Escape The Witch est un escape game en Bash dans lequel le joueur doit s'échapper en résolvant des énigmes et en avançant de pièce en pièce.

Le principe est de trouver les bonnes commandes, d'utiliser les scripts fournis et de récupérer les indices nécessaires pour sortir de la maison de la sorcière.
