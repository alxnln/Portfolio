# Licence Info UP8

Ce dossier regroupe mes trois projets réalisés dans le cadre de la Licence Info UP8.

## Projets

### Escape The Witch
Un escape game en Bash dans lequel il faut s'échapper en résolvant des énigmes et en avançant de pièce en pièce.

### Maze Solver
Un projet de résolution de labyrinthe basé sur une recherche DFS.

### Sutom Game
Un jeu de devinette de mots en C avec essais limités, affichage coloré et dictionnaire.
