# Ecole 42

Ce dossier regroupe quelques projets que j'ai réalisés dans le cadre de la formation Ecole 42.

## Projets

### Libft
Une bibliothèque personnelle en C qui réimplémente des fonctions utiles de la librairie standard, avec une partie bonus sur les listes chaînées.

### ft_printf
Une réimplémentation de `printf` en C avec gestion des conversions principales et des cas particuliers classiques.

### Minishell
Un shell simplifié en C qui reproduit des comportements essentiels de Bash comme l'exécution de commandes, les pipes, les redirections et les builtins.
